# Retail Sales Intelligence Project

End-to-end beginner data project combining SQL, Python and Power BI.

## Project Overview

This project analyzes retail sales data using SQL queries, Python data checks and a Power BI dashboard.

## Tools Used

- SQL
- Python
- Power BI
- CSV data
- GitHub

## Project Workflow

1. Analyze sales data with SQL
2. Check and clean data with Python
3. Build an interactive Power BI dashboard
4. Document results on GitHub

## Key Analysis

- Total sales
- Average sales
- Sales by category
- Sales by region
- Top products

## Files

- data/sales_data.csv
- sql/analysis_queries.sql
- python/data_cleaning.py
- dashboard/dashboard_screenshot.png

## Learning Goal

Practice a realistic data workflow from raw data to business insights.
